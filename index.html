Virtual Breathing:

Cellular Respiration and Photosynthesis

Use the space below to see how our app works before jumping in

  1. Placing the mouse over an element will expand it for easier viewing.

  2. Click and hold on the elements below to start dragging them around.

  3. While dragging, place an element over another element and release to swap them.

  4. If an element can be placed, the item will visually rotate/tilt so you know the item is droppable.

  5. If an element is dropped outside of acceptable zones the element will return to its previous position.
