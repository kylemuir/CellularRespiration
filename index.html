Virtual Breathing:

Cellular Respiration

Use the space below to see how our app works before jumping in

  1. Placing the mouse over an element will expand it for easier viewing

  2. Click on the elements below to start dragging them

  3. While dragging place it over another element and release to swap the two elements

  4. If an element can be placed the item will visual rotate/tilt so you can tell it's droppable

  5. If an element is dropped outside of acceptable zones it will return to previous position
